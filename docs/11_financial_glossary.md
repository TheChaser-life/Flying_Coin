# 📖 Thuật Ngữ Tài Chính & Chứng Khoán

Danh sách các thuật ngữ cần tìm hiểu để xây dựng hệ thống Market Data Mining & Forecasting.

---

## 1. Dữ Liệu Giá (Price Data)

| Thuật ngữ | Tiếng Việt | Giải thích |
|-----------|-----------|-----------|
| **OHLCV** | Mở-Cao-Thấp-Đóng-KL | Open, High, Low, Close, Volume — dữ liệu giá cơ bản |
| **Open** | Giá mở cửa | Giá đầu tiên của phiên giao dịch |
| **High** | Giá cao nhất | Giá cao nhất trong phiên |
| **Low** | Giá thấp nhất | Giá thấp nhất trong phiên |
| **Close** | Giá đóng cửa | Giá cuối cùng của phiên |
| **Adjusted Close** | Giá đóng cửa điều chỉnh | Giá đóng cửa đã tính cổ tức, chia tách |
| **Volume** | Khối lượng giao dịch | Số lượng cổ phiếu/coin được giao dịch |
| **Market Cap** | Vốn hóa thị trường | Giá × Tổng số cổ phiếu lưu hành |
| **Candlestick** | Biểu đồ nến | Biểu đồ hiển thị OHLC bằng hình nến |
| **Timeframe** | Khung thời gian | Khoảng thời gian mỗi nến: 1m, 5m, 1h, 1D, 1W |
| **Ticker / Symbol** | Mã chứng khoán | Mã viết tắt (VD: AAPL = Apple, BTC = Bitcoin) |
| **Bid / Ask** | Giá mua / Giá bán | Bid = giá người mua sẵn sàng trả, Ask = giá người bán yêu cầu |
| **Spread** | Chênh lệch giá | Khoảng cách giữa Bid và Ask |
| **Order Book** | Sổ lệnh | Danh sách tất cả lệnh mua/bán đang chờ |

## 2. Chỉ Báo Kỹ Thuật (Technical Indicators)

| Thuật ngữ | Giải thích | Dùng để |
|-----------|-----------|---------|
| **SMA** (Simple Moving Average) | Trung bình động đơn giản | Xác định xu hướng |
| **EMA** (Exponential Moving Average) | Trung bình động hàm mũ (nặng dữ liệu gần) | Xu hướng nhạy hơn SMA |
| **RSI** (Relative Strength Index) | Chỉ số sức mạnh tương đối (0-100) | Overbought (>70) / Oversold (<30) |
| **MACD** (Moving Average Convergence Divergence) | Hội tụ/Phân kỳ trung bình động | Tín hiệu mua/bán khi MACD cắt signal line |
| **Bollinger Bands** | Dải Bollinger (SMA ± 2σ) | Đo độ biến động, xác định vùng giá |
| **ATR** (Average True Range) | Phạm vi dao động trung bình | Đo mức biến động |
| **ADX** (Average Directional Index) | Chỉ số xu hướng trung bình | Đo sức mạnh xu hướng (>25 = mạnh) |
| **Stochastic Oscillator** | Dao động ngẫu nhiên | Xác định overbought/oversold |
| **OBV** (On-Balance Volume) | Khối lượng cân bằng | Xác nhận xu hướng qua volume |
| **VWAP** (Volume Weighted Avg Price) | Giá trung bình theo khối lượng | Benchmark cho giao dịch |
| **Parabolic SAR** | Stop And Reverse | Xác định điểm đảo chiều |
| **Ichimoku Cloud** | Mây Ichimoku | Xu hướng, hỗ trợ, kháng cự trong 1 indicator |

## 3. Phân Tích & Dự Báo (Analysis & Forecasting)

| Thuật ngữ | Giải thích |
|-----------|-----------|
| **Technical Analysis** | Phân tích kỹ thuật — dùng biểu đồ và indicators để dự đoán giá |
| **Fundamental Analysis** | Phân tích cơ bản — dựa trên tài chính công ty (EPS, P/E, Revenue) |
| **Sentiment Analysis** | Phân tích tâm lý — đánh giá cảm xúc thị trường từ tin tức/social media |
| **Time Series** | Chuỗi thời gian — dữ liệu có thứ tự thời gian |
| **Trend** | Xu hướng — hướng đi chung của giá (tăng/giảm/đi ngang) |
| **Support** | Ngưỡng hỗ trợ — mức giá mà cầu đủ mạnh để ngăn giá giảm |
| **Resistance** | Ngưỡng kháng cự — mức giá mà cung đủ mạnh để ngăn giá tăng |
| **Breakout** | Phá vỡ — giá vượt qua support/resistance |
| **Pullback** | Điều chỉnh — giá giảm tạm thời trong xu hướng tăng |
| **Bull Market** | Thị trường tăng — giá tăng liên tục, lạc quan |
| **Bear Market** | Thị trường giảm — giá giảm liên tục, bi quan |
| **Sideways** | Đi ngang — giá dao động trong vùng hẹp |
| **Fear & Greed Index** | Chỉ số nỗi sợ & lòng tham — đo cảm xúc thị trường (0-100) |

## 4. Quản Lý Danh Mục (Portfolio Management)

| Thuật ngữ | Giải thích |
|-----------|-----------|
| **Portfolio** | Danh mục đầu tư — tập hợp các tài sản đang nắm giữ |
| **Asset Allocation** | Phân bổ tài sản — tỷ trọng mỗi tài sản trong danh mục |
| **Diversification** | Đa dạng hóa — phân tán rủi ro bằng cách đầu tư nhiều tài sản |
| **Rebalancing** | Tái cân bằng — điều chỉnh tỷ trọng về mức mục tiêu |
| **Expected Return** | Lợi nhuận kỳ vọng — lợi nhuận dự kiến trung bình |
| **Volatility** | Biến động — độ dao động giá, đo bằng standard deviation |
| **Sharpe Ratio** | Tỷ lệ Sharpe = (Return - Risk-free) / Volatility. Cao hơn = tốt hơn |
| **Max Drawdown** | Mức sụt giảm tối đa — % giảm lớn nhất từ đỉnh đến đáy |
| **Efficient Frontier** | Đường biên hiệu quả — tập hợp danh mục tối ưu (Markowitz) |
| **Risk-free Rate** | Lãi suất phi rủi ro — thường dùng lãi suất trái phiếu chính phủ |
| **Beta** | Hệ số Beta — độ nhạy của tài sản so với thị trường chung |
| **Alpha** | Hệ số Alpha — lợi nhuận vượt trội so với benchmark |
| **Correlation** | Tương quan — mức độ 2 tài sản di chuyển cùng/ngược nhau (-1 → +1) |
| **Covariance** | Hiệp phương sai — đo mức độ biến động cùng nhau |

## 5. Giao Dịch (Trading)

| Thuật ngữ | Giải thích |
|-----------|-----------|
| **Long** | Mua — mua tài sản với kỳ vọng giá tăng |
| **Short** | Bán khống — vay tài sản bán, mua lại khi giá giảm |
| **Buy Signal** | Tín hiệu mua — indicator/model gợi ý nên mua |
| **Sell Signal** | Tín hiệu bán — indicator/model gợi ý nên bán |
| **Stop Loss** | Cắt lỗ — lệnh tự động bán khi giá giảm đến ngưỡng |
| **Take Profit** | Chốt lời — lệnh tự động bán khi giá tăng đến mục tiêu |
| **Backtesting** | Kiểm tra ngược — test chiến lược trên dữ liệu lịch sử |
| **Paper Trading** | Giao dịch giả lập — trading bằng tiền ảo để thử nghiệm |
| **Slippage** | Trượt giá — chênh lệch giữa giá kỳ vọng và giá thực tế |
| **Liquidity** | Thanh khoản — khả năng mua/bán nhanh mà không ảnh hưởng giá |

## 6. Machine Learning cho Tài Chính

| Thuật ngữ | Giải thích |
|-----------|-----------|
| **RMSE** | Root Mean Square Error — sai số dự báo (thấp hơn = tốt hơn) |
| **MAE** | Mean Absolute Error — sai số tuyệt đối trung bình |
| **MAPE** | Mean Absolute Percentage Error — sai số phần trăm |
| **Directional Accuracy** | Tỷ lệ dự đoán đúng chiều (tăng/giảm) |
| **Walk-Forward Validation** | Validation tuần tự — train trên quá khứ, test trên tương lai gần |
| **Feature Engineering** | Kỹ thuật đặc trưng — tạo biến mới từ dữ liệu thô |
| **Overfitting** | Quá khớp — model chỉ tốt trên train data, tệ trên test data |
| **Ensemble** | Kết hợp — gộp nhiều models để tăng accuracy |
| **Hyperparameter Tuning** | Tinh chỉnh siêu tham số — tìm cấu hình tối ưu cho model |
| **Model Drift** | Trôi mô hình — accuracy giảm theo thời gian do thị trường thay đổi |

## 7. NLP cho Tài Chính

| Thuật ngữ | Giải thích |
|-----------|-----------|
| **Sentiment Score** | Điểm cảm xúc — từ -1.0 (tiêu cực) đến +1.0 (tích cực) |
| **FinBERT** | BERT fine-tuned cho tài chính — phân tích sentiment tin tức |
| **Tokenization** | Tách từ — chia văn bản thành các token |
| **Stop Words** | Từ dừng — các từ phổ biến không mang ý nghĩa (và, là, của...) |
| **Lemmatization** | Đưa từ về dạng gốc (running → run) |
| **Named Entity Recognition** | Nhận dạng thực thể — xác định tên công ty, mã CK trong text |
| **Word Embedding** | Vector hóa từ — biểu diễn từ bằng vector số |

## 8. Kinh Tế Vĩ Mô (liên quan đến dự báo)

| Thuật ngữ | Giải thích | Ảnh hưởng đến thị trường |
|-----------|-----------|-------------------------|
| **CPI** | Consumer Price Index — chỉ số giá tiêu dùng | CPI tăng → lo ngại lạm phát → thị trường giảm |
| **GDP** | Gross Domestic Product — tổng sản phẩm quốc nội | GDP tăng → kinh tế tốt → thị trường tăng |
| **Interest Rate** | Lãi suất — lãi suất do ngân hàng trung ương quyết định | Lãi suất tăng → chi phí vay tăng → thị trường giảm |
| **Inflation** | Lạm phát — mức tăng giá hàng hóa chung | Lạm phát cao → FED tăng lãi suất |
| **Fed / FOMC** | Cục Dự trữ Liên bang Mỹ | Quyết định lãi suất, ảnh hưởng toàn cầu |
| **Yield Curve** | Đường cong lợi suất — lãi suất trái phiếu theo kỳ hạn | Inverted = báo hiệu suy thoái |
| **QE** | Quantitative Easing — nới lỏng định lượng | FED bơm tiền → thị trường tăng |
| **QT** | Quantitative Tightening — thắt chặt định lượng | FED rút tiền → thị trường giảm |
